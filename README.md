# Script Studio

> 抖音/小红书口播文案语义级合规审核 + 画面设计工作台

## 📋 项目概述

Script Studio 是一个面向**短视频内容创作**的工作台，提供：
- 🎬 **口播文案审核** - 语义级合规检测，避免违规内容
- 🎨 **画面设计工作台** - 分镜脚本设计与素材管理
- 📊 **内容管理** - 多平台（抖音/小红书）内容统一管理

## ✨ 核心功能

### 文案审核
- 语义级违规检测（不只是关键词匹配）
- 多平台规则适配（抖音、小红书）
- 审核报告生成

### 画面设计
- 分镜脚本设计
- 素材库管理
- 画面预览与导出

### 内容管理
- 项目/素材统一管理
- 多格式导出（HTML、JSON）
- 团队协作

## 🏗️ 技术栈

| 技术 | 说明 |
|------|------|
| **HTML/CSS/JS** | 前端界面 |
| **Python** | 后端脚本/数据处理 |
| **Node.js** | 工具脚本 |

## 🚀 快速开始

### 环境要求

- 现代浏览器（Chrome、Edge、Firefox）
- Python 3.8+（可选，用于后端脚本）

### 使用方法

1. 克隆项目
   ```bash
   git clone https://github.com/Panda-Lorrain/script-studio.git
   ```

2. 打开工作台
   ```bash
   cd script-studio
   # 直接用浏览器打开
   start workspace/index.html
   # 或者用本地服务器
   python -m http.server 8080
   ```

3. 开始使用
   - 创建新项目
   - 上传/编写口播文案
   - 运行合规审核
   - 设计分镜画面

## 📝 使用场景

### 场景 1：口播文案审核
1. 在工作台输入或粘贴文案
2. 选择目标平台（抖音/小红书）
3. 点击"审核"，查看语义级检测结果
4. 根据建议修改文案

### 场景 2：分镜脚本设计
1. 创建新项目
2. 添加分镜场景
3. 配置素材和画面
4. 导出 HTML 分镜脚本
